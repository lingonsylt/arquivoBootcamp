Here I keep some java exercises.

Note: these exercises were made to practice some programming logic concepts such as conditionals, loops, arrays and so on. __*They're not meant to be the most efficient and lean programs*__ they were to focus particularly on just one aspect of programming logic though. I tried as much as possible in each case to stick to this given one aspect mainly looking forward to master it.

## basicSequential/

In here you'll find some basics i/o handling. I wrote some simple programs that read inputs from the user, process it and then print the output on the terminal.

## basicConditional/

In here you'll find some basics conditionals: _if-else_, _if else-if_ and _switch cases_. Again I wrote some simple programs that read input from the user, process it  and then print the output on the terminal.

## basicLoop/

In here you'll find basics loops: _while_, _do-while_ and _for_. As mentioned above these programs also read and print data from the user

# Java/

Aqui mantenho alguns exercício de java.

Nota: estes exercícios foram feitos para que treinasse alguns conceitos de lógicas de programação como laços condicionais, laços de repetição, vetores, etc. _*Eles não foram feitos com o intento de serem o mais eficiênte e enxutos*_, ao contrário eles foram feitos com foco em determinados aspectos da lógica de programação. Eu tentei ao máximo me manter apenas em um destes aspectos em cada caso no intuito de dominá-los.

## basicSequential/

Aqui você encontra entradas e saídas de dados básicas. Escrevi alguns programas simples que leem  dados do usuário, os processam e depois os escrevem no terminal.

## basicConditional/

Aqui você encontra laços condicionais básicos: _se-senão_, _se senão-se_ e _enquanto-casos_. De novo, escrevi alguns programas simples que leem dados do usuário, os processam e os escrevem no terminal.

## basicLoop/

Aqui você encontra laços de repetição básicos: _enquanto_, _faça-enquanto_ e _para_. Como mencionado acima, estes programas também leem e escrevem dados dados pelo usuário.

## basicArraynMatrix/

Finalmente, aqui você encontra vetores e matrizes básicas. O que já foi dito acima também serve aqui.
